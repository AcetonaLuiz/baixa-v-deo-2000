# baixa-v-deo-2000

#  Baixador do YouTube em .bat

Um script simples em Batch (.bat) para Windows que permite baixar vídeos e áudios do YouTube mantendo qualidade usando **yt-dlp** e **ffmpeg**

##  Pré-requisitos

Para utilizar o script, você precisa ter o **yt-dlp** e o **ffmpeg** instalados no seu sistema. 

Você pode instalá-los rapidamente via Prompt de Comando (CMD) executando:

```cmd
winget install --id yt-dlp.yt-dlp
winget install --id Gyan.FFmpeg
